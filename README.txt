Esse projeto é um estudo com o modulo Sequilize (Node.js), o qual faz conectar o codigo com o banco de dados(MySQL)

para funcionar, requer instalar:

    ## npm i --save body-parser
    ## npm i --save Sequilize
    ## npm i --save mysql2
